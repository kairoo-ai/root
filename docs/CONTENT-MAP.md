## 2. Proposed proper-product IA

```
PUBLIC (marketing shell)
/                  Home - tightened hero + proof + pillars + featured tools + pricing teaser + CTA
/features          Hub (3 pillars)
  /features/career    33 career tools (from registry)
  /features/learning  5 learning features (+ coming-soon badges)
  /features/teams     Team/Enterprise analytics + Team Skill Matrix radar
  /features/[tool]    (optional) per-tool detail from registry
/pricing           Full pricing (3 tiers) + objections + trust
/how-it-works      Sense→Think→Act + AI-integration explainer (NEW; from architecture story)
/customers         Testimonials + impact metrics + anonymized ICP use-cases (NEW)
/about             Mission/team (+ public-safe primary value prop)
/contact           Contact + demo scheduling
/security          Public trust: SOC2/GDPR/HIPAA/ISO + security layers + perf targets (NEW)
/style             Component gallery (noindex, exists)
LEGAL              /privacy /terms /cookies /acceptable-use /ai-disclosure /dpa /sub-processors

INVESTORS (gated shell - requiresAuth)
/investors             Overview hub + headline raise metrics
/investors/deck        Opportunity, 10-slide deck, forecast, financial table, 90-day MVP, contact
/investors/market      Sizing, TAM/SAM/SOM, drivers, competitive matrix+chart, GTM, 90-day growth
/investors/strategy    Validation, problem analysis, value props, ICPs, MoSCoW, roadmap
/investors/architecture Full technical blueprint (gated)

(app)/   Reserved authed shell - future home where the 35 ready tools run
```

**Principle:** investor-only hard numbers (raise, MRR/churn projections, equity, validation score, financial table, ICPs, MoSCoW) stay **gated** under `/investors/*`. Public-safe proof (testimonials, impact %, pricing, tool catalog, compliance) goes to public marketing pages.
