# Kairoo — Brand Assets (backup & reference)

Permanent, committed copies of the brand identity decided during the 2026-06-13 rebrand
brainstorm. The live design doc is
[`../superpowers/specs/2026-06-13-rebrand-and-legal-design.md`](../superpowers/specs/2026-06-13-rebrand-and-legal-design.md).

> Why this folder exists: the interactive brainstorm ran under `.superpowers/brainstorm/`,
> which is gitignored and gets cleaned up. These files are the durable backup.

## Name
**Kairoo** — from Greek *kairos*, "the opportune moment." Story: *the right moment to grow.*

## Logo / glyph (locked variant "B3")
The owner's original arc sweep + a small curved stem (reads as **K** on a second look) +
**two teal dots**: a small "waypoint" on the journey and the larger "o" / *kairos* **moment**
dot. All curves — no straight/edgy lines.

| File | Use |
|------|-----|
| [`kairoo-glyph.svg`](kairoo-glyph.svg) | Primary, light backgrounds (navy stroke, teal dots) |
| [`kairoo-glyph-dark.svg`](kairoo-glyph-dark.svg) | Dark backgrounds (white stroke, bright-teal dots) |
| [`kairoo-glyph-mono.svg`](kairoo-glyph-mono.svg) | One-color (`currentColor`) for stamps/emboss |
| [`kairoo-favicon-16.svg`](kairoo-favicon-16.svg) | Smallest favicon — waypoint dropped, stroke thickened |

**Wordmark lockup:** glyph + "Kairoo" set bold, ~ -1.5px letter-spacing.
**Favicon:** thicken stroke as size shrinks (8.05 → ~11 at 16px). Use the simplified 16 file for the tiniest size.

## Color system
| Token | Hex | Role |
|-------|-----|------|
| Navy | `#0B1F3A` | Anchor — text, enterprise tier, trust |
| Teal | `#0D9488` | Primary action (light) |
| Teal-Bright | `#2DD4BF` | Primary action (dark) |
| Amber | `#F59E0B` | Warmth + Free tier |
| Gold | `#CBA34A` | Enterprise-premium accent (sparing) |
| Mist | `#F8FAFC` | Light background |

**Tier accents:** Free = Amber · Pro = Teal · Enterprise = Navy + Gold.
**Dark theme:** navy-derived (bg `#071426`, surface `#0F2740`), NOT pure black.
